# Customer Purchase Behaviour Analysis

## Overview

This project demonstrates an end-to-end Data Analytics workflow using Python, MySQL, and Power BI. The goal is to analyze customer purchasing behavior, identify key business trends, and present actionable insights through an interactive dashboard.

The project covers:

* Data loading and preprocessing using Python
* Exploratory Data Analysis (EDA)
* Data cleaning and transformation
* SQL analysis using MySQL
* Interactive dashboard development in Power BI

---

## Dataset

The dataset contains customer demographic and purchasing information, including:

* Customer ID
* Age Group
* Gender
* Region
* Product Category
* Purchase Amount
* Purchase Frequency
* Loyalty Status
* Income Level

## Key Insights

* Electronics is the highest revenue-generating product category.
* Customers aged 26–35 contribute the largest share of total spending.
* Rare purchasers represent the largest customer segment.
* Regular loyalty members account for over 60% of customers.
* Clothing and Food categories are strong revenue contributors after Electronics.
* Beauty and Home categories generate comparatively lower revenue.
